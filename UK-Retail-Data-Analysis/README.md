# Retail Sales Analysis (R Project)

## Project Overview

This project analyzes retail transaction data to generate insights into product performance, customer behavior, geographic trends, and seasonal patterns.

## Tools & Technologies

R (dplyr, ggplot2)

Data Cleaning & Transformation

Data Visualization

 ## Key Analysis Performed

Removed cancelled invoices and cleaned dataset

Calculated product-level revenue

Identified Top 10 and Bottom 10 products

Analyzed high-value customers

Conducted country-wise revenue analysis

Performed monthly trend and seasonality analysis

## Key Insights

Revenue is diversified; top 10 products contribute ~10.8%

A small group of customers contributes a significant portion of revenue

UK dominates revenue, indicating geographic concentration

Strong seasonality observed during holiday periods

Certain products consistently underperform
